# envs



## Description

Flexible git branch manager for collaborative environment.

## Usage

```
envs <appname>  <command>  [branch]  <env>
    

Command:
    - deploy
    - remove
    - reset
    - list


update

```

## Install

To install, use `go get`:

```bash
$ go get -d github.com/zafarella/envs
```
